```markdown
ViperHighlighter — Dalamud plugin (テンプレート)

目的:
- 指定した効果ID（Effect ID）が付与されているターゲットに対して、
  背面（背裂/背撃）または側面（側裂/側撃）を扇形で強調表示します。

デフォルト効果ID（ユーザー提供）:
- 背面系: 4093, 4094
- 側面系: 4095, 4096

使い方:
1. ソースをビルドして DLL を生成します。
2. manifest.json と DLL を Dalamud のプラグインフォルダに置き、プラグインを有効化します。
3. プラグイン設定（Dalamud で ViperHighlighter の設定を開く）で色・角度・距離や効果IDを調整できます。

注意:
- Dalamud / FFXIV の API バージョン差により一部プロパティ名が異なる可能性があります。コンパイル時にエラーが出たら、そのエラーメッセージを送ってください。API 名に合わせて修正します。
```
